# Custom style

If you want to completely customize the style,you need a reference [lib/index.css](../lib/index.css),and using the props.themes props.prefixClassName.

````css
html .alert--themes-info {
    border-color: #aaa;
    background-color: #eee;
    color:#333;
}
````
