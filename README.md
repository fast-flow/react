# react-module

## Installation

```shell
npm i -g yo                     --registry=https://registry.npm.taobao.org
npm i -g fis3                   --registry=https://registry.npm.taobao.org
npm i -g generator-react-module --registry=https://registry.npm.taobao.org
```

```shell
mkdir app && cd app
yo react-module
```

## Usage

### development

```shell
npm run hot
npm run dev
```

### publish gh-pages
```shell
npm run gh
npm run gh-push
```

### publish npm

```shell
npm run npm
cd output && npm publish
```
